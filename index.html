Project 2

Cycloneopoly is an Android based app I developed as part of a team of four. This app was made to be a mobile version of the board game Monopoly with an Iowa State University theme. We worked to complete this project over the span of a semester as a part of our classwork for COM S 309. The goal was to be able to make a fully function app that could operate out of a server and run multiple different games at any one time. The app would also be able to run both multiplayer as pass and play, with bots or across the server.
